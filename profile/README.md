## What is Handoff?

**Handoff** lets you chat with Claude, GPT, Gemini, Mistral, DeepSeek, Groq and more — switching models mid-conversation without losing context. Your API keys, your data.

- **[handoff.cl](https://handoff.cl)** — Web app. Use any model, share threads, manage projects.
- **[VS Code Extension](https://github.com/handoffcl/handoff-extension)** — Agent with filesystem, git, and terminal access.
- **[Handoff Coder](https://github.com/handoffcl/handoff-coder)** — Open source Modelfile for spec-first development.

## Handoff Coder — Open Source

```
Regla central: si el costo del proceso supera el riesgo del cambio,
usa el nivel más liviano posible.

directo   → responde de inmediato
contextual → lee contexto, responde  
mini-flow  → propón en 1 línea, espera ok
full-flow  → analiza → propón → espera ok
```

The Modelfile behind Handoff Coder is public. Use it, improve it, contribute.
